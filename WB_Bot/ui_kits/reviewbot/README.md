# ReviewBot UI Kit

## Overview
High-fidelity recreation of the ReviewBot web app — an AI-powered marketplace review response tool for Wildberries (WB) and Ozon.

## Files
- `index.html` — Full interactive prototype (React SPA)
- `Header.jsx` — App header with platform switcher
- `ReviewCard.jsx` — Review card component
- `RespondPanel.jsx` — Response editor panel
- `SetupForm.jsx` — Brand setup screen

## Design Width
700px max content (mobile-first, responsive)

## Screens
1. **Setup** — First-run brand configuration
2. **Review List** — Pending and responded reviews, WB/Ozon tabs
3. **Respond** — AI generation + edit + send
4. **Success** — Confirmation screen

## Notes
- Uses system font stack (Inter as Google Fonts substitute)
- Two platform themes: WB purple, Ozon blue
- All API calls are mocked for prototype use
